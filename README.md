# Tsembilan Keyboard

Keyboard 12 tombol sederhana untuk mengetik di Android. Ditargetkan untuk kemudahan mengetik klasik (multi-tap) dan dirancang untuk kompatibilitas penuh dengan kebutuhan pengiriman pesan singkat (seperti WhatsApp).

## Fitur Utama
- Multi-tap klasik 12 tombol layaknya handphone lawas.
- Tidak ada auto-correct, tidak ada prediksi kata.
- Ringan dan tidak memberatkan sistem.
- Dirancang untuk kenyamanan pada mode portrait.

## Privacy & Security

Tsembilan Keyboard dibangun dengan fokus mutlak pada privasi pengguna:
- **Tidak memakai permission INTERNET**
- **Tidak mengirim data ke server**
- **Tidak menyimpan teks yang diketik**
- **Tidak menggunakan analytics**
- **Tidak menggunakan iklan**
- **Tidak membaca SMS, kontak, notifikasi, lokasi, atau clipboard**

Semua proses pengetikan murni terjadi secara lokal di perangkat Anda.

## Instalasi dan Aktivasi
Setelah menginstal aplikasi, buka Tsembilan Keyboard dari layar utama (Launcher). Anda akan dihadapkan pada layar aktivasi.
1. Klik **Aktifkan Keyboard** untuk mengaktifkan Tsembilan Keyboard di pengaturan sistem Android.
2. Klik **Pilih Keyboard** untuk beralih menggunakan Tsembilan Keyboard.

*(Keyboard harus diaktifkan oleh user secara manual lewat pengaturan Android)*
